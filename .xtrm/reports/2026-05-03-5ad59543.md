## Code Changes

### gzrx infrastructure (Phases 1-7 + fixes)

**Catalog files** (`.specialists/catalog/`):
- `index.json` — entry point with `precedence_order: [native, gitnexus, serena]`, lists all 3 catalogs with package/version metadata for drift checks
- `native.json` — 7 native tools (read/grep/find/ls/bash/edit/write) split per tier
- `gitnexus.json` — 7 gitnexus_* tools per tier
- `serena.json` — 43 Serena tools per tier matching source-tier policy in pi/session.ts:164-214

**Resolver library** (`src/specialist/manifest-resolver.ts`, 188 lines):
- Pure function `resolveManifestTools(input)` returns `{tools, toolsList, deniedNatives, deniedNativesMode, preferenceSignals, downgradeReasons, warnings, attribution}`
- Six precedence layers per design §3.0: catalog → tier_policy → manifest → specialist_override → specialist_exclusion → runtime_health
- Hard-deny restoration when replacement extension is unhealthy/unknown/catalog-incompatible
- `LEGACY_PERMISSION_TOOL_STRINGS` exported for byte-equivalence snapshot tests

**Diagnostic CLI** (`src/specialist/resolution-diagnostics.ts` + `src/cli/config.ts`):
- `sp config show <specialist> --resolved` reads specialist JSON, probes extension health via global node_modules, applies resolver, prints effective manifest + per-layer attribution + extension state + drift + final --tools
- Tier read from `execution.permission_required` (was hardcoded HIGH initially)
- Per-specialist override loaded from `permissions[tier]` block in specialist JSON

**Runtime threading** (`src/pi/session.ts:243-269`):
- `loadSharedToolCatalogIndex()` cached catalog loader
- `resolvePermissionTools(level)` returns resolver-derived `--tools` string
- `probeExtensionHealth(packageName)` checks global node_modules for `package.json` presence
- Activated by `useSharedToolResolver` option OR `SPECIALISTS_USE_RESOLVER=1` env var
- Default OFF — legacy `mapPermissionToTools` still active

**Per-specialist override** (`config/specialists/explorer.specialist.json`):
- Added `permissions.READ_ONLY.denied_natives_when_extension: ["grep","find","ls"]` with `denied_natives_mode: "hard"`
- **Only specialist migrated** — all others use tier defaults
